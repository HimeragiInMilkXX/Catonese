# Catonese（キャトニーズ）

Catonese は、単語や文法の暗記ではなく、**実際に使える会話と発音**にフォーカスした語学学習プラットフォームです。  
日常会話の実践とスピーキング・リスニング力の強化を重視して設計しています。

## デモ動画

学習フローと機能の概要は、以下の動画でご覧いただけます（1.5倍速推奨）：  
https://youtu.be/PX7nrqomIBs

## 使用技術

- フロントエンド：Nuxt.js, TypeScript, Tailwind CSS, shadcn/vue, SCSS  
- バックエンド：Spring Boot, Python（アラインメント処理）  
- デザイン：Figma  

## ユーザー向け機能

- 初心者向けの日常会話をベースにした、追いやすいレッスン構成  
- 文構造ごとに文法パターンを学べる動的ワイルドカード（名詞・動詞など）  
- フラッシュカードと四択などの問題形式による復習機能  
- 発音の自信度・正確さ・リアルタイムフィードバックを返す音声認識機能  
- 音声認識は動的な名詞・動詞ワイルドカードにも対応  
- 語彙・文法は会話の中で自然に登場し、**スピーキング／リスニングを常に最優先**  
- すべてのデバイスで学習できるレスポンシブデザイン  

## 開発者向け機能

- レッスンはサーバー上の JSON ファイルとして管理されており、  
  新規作成・変更はコードを書かずに追加可能なスケーラブル設計  
- フロントエンドは新しいレッスンを自動で読み込み、ユーザーの進捗も自動でトラッキング  
  開発者側はデータベースさえ用意すれば運用できる構成になっています。
